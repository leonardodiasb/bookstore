# Bookstore

> A Bookstore website where it's possible to add, remove a book and the book list is displayed. The app was created using React and Redux.

## Built With

- HTML/CSS
- Javascript
- React
- Redux
- NPM

## Getting Started

### Install

- `git clone https://github.com/leonardodiasb/bookstore.git`
- `cd bookstore`
- `npm install`
- `npm install axios`
- `npm start`
